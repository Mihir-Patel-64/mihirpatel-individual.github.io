# Resources

This page provides all downloadable project resources used in the development, testing, and final delivery of the Smart Curtain Control subsystem. These files include the complete MPLAB X firmware project, CAD models for any fabricated components, and other supporting materials required for replication or further development of the design.

All resources are packaged in standardized .zip archives to ensure compatibility, portability, and ease of access across different operating systems and design environments.
